# Quick Build Guide

## Local Development (Your Machine)
```bash
flutter build apk --release
# Find APK at: build/app/outputs/flutter-apk/app-release.apk
```

## GitHub Actions (Automated)
- Push code to GitHub
- CI/CD builds APK automatically
- Download from Actions artifacts
