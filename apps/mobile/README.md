# Khatabook Pro - Mobile

A high-performance Flutter mobile application for financial tracking.

## Tech Stack

- **Framework**: Flutter / Dart
- **State Management**: Provider
- **Local Storage**: sqflite
- **Networking**: Dio

## Local Setup

1. `cd apps/mobile`
2. `flutter pub get`
3. `flutter run`

## Building APK

This project is configured with GitHub Actions to build APKs automatically on push.
Check the `.github/workflows/android_build.yml` for the build pipeline details.
